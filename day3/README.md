# 使用npm安装第三方模块

如果你这个电脑已经安装了nodejs环境，那你就可以在命令行里面使用`npm`下载一些第三方模块
```js
npm i [模块]
npm install [模块]
```

# request

在命令行里面使用`request`
```js
npm i request
```
一般安装成功的话，你就会在文件夹里面看到`node_modules`

# express

先安装express第三方模块依赖包
```js
npm i express
```
