# Node.js

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境

本质谷歌浏览器的控制台

下载[nodejs](http://nodejs.cn/download/)的安装包,直接点击默认安装

安装完成后，在命令行里面，输入,如果有版本号出现，那证明安装成功

```bash
node -v
```
